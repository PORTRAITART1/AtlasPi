🎯 FONCTIONS CRÉÉES:

1. renderAdminCounters(counts)
   • Génère le HTML pour les 4 cartes
   • Affiche les nombres
   • Applique les classes CSS

2. countStatusesFromListings()
   • Parcoure les listings visibles
   • Compte par statut
   • Retourne {pending_review: N, approved: N, ...}

3. updateAdminCounters()
   • Principale fonction de mise à jour
   • Appelle countStatusesFromListings()
   • Appelle renderAdminCounters()

═══════════════════════════════════════════════════════════════════

⚙️ LOGIQUE CHOISIE:

Type: Comptage à partir des données déjà chargées
Raison: Simplicité, pas d'API supplémentaire, performance

Flux:
Backend retourne listings
  ↓
Listings affichés dans le DOM
  ↓
updateAdminCounters() → countStatusesFromListings()
  ↓
Compte les statuts visibles
  ↓
renderAdminCounters() affiche les cartes colorées

═══════════════════════════════════════════════════════════════════

📊 EXTRAITS DE CODE:

Fonction Principale:
───────────────────────────────────────────────────────────────────
function updateAdminCounters() {
  const pendingListings = document.querySelectorAll('#pendingListingsList > div');
  if (pendingListings.length === 0) {
    renderAdminCounters({
      pending_review: 0,
      approved: 0,
      rejected: 0,
      suspended: 0
    });
    return;
  }
  const counts = countStatusesFromListings();
  renderAdminCounters(counts);
}
───────────────────────────────────────────────────────────────────
