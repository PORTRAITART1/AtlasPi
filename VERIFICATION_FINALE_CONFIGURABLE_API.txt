CONFIGURATION À RUNTIME:

Admin peut changer l'API_BASE_URL sans redémarrage:

1. Ouvrir Developer Console (F12)
2. Taper:
   window.ATLASPI_CONFIG.setApiBaseUrl("https://new-api-url.com");
3. L'URL est sauvée en localStorage
4. Prochaine requête utilise la nouvelle URL
5. Pour reset:
   window.ATLASPI_CONFIG.resetApiBaseUrl();
