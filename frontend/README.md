# AtlasPi Ultra Premium Homepage

AtlasPi is a premium landing page for the global Pi community.

## Files
- index.html
- style.css
- script.js
- privacy.html
- terms.html

## Description
AtlasPi helps pioneers discover trusted places and services where Pi can be used simply, locally, and confidently.

## Deploy with GitHub Pages
1. Upload these files to your `atlaspi` GitHub repository.
2. Enable GitHub Pages from the `main` branch.
3. Your public site will be available at:
   https://portraitart1.github.io/atlaspi/
