📈 USAGE FLOW
───────────────────────────────────────────────────────────────────────────────────

1. Admin opens AtlasPi → Portal loads normally

2. Admin clicks "Show Admin Moderation" → Admin section appears
   • Counters display at top: 0 | 0 | 0 | 0 (waiting for data)

3. Admin enters secret key and clicks "Load Pending Listings"
   • Backend returns listings
   • Counters auto-update: e.g., 5 | 3 | 1 | 1

4. Admin reviews and moderates listings
   • Changes status (e.g., pending → approved)
   • Clicks "Apply Moderation"
   • Page reloads automatically
   • Counters recalculate: 4 | 4 | 1 | 1

5. Repeat step 4 as needed
   • Counters always stay in sync
